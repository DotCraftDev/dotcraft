# User Profile

## Primary User

- Role: experienced software engineer or tech lead
- Language: English
- Preference: direct, high-signal explanations
- Values: correctness, maintainability, and explicit tradeoffs

## Project Preferences

- Prefer references to concrete files, symbols, and configuration keys
- Prefer production-safe recommendations
- Prefer backward-compatible changes unless otherwise requested

## Communication Preferences

- Avoid over-explaining basics
- Surface risks and assumptions early
- Use examples only when they clarify the point
