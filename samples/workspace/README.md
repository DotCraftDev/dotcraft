# Workspace

This folder can be used directly as the DotCraft workspace when cloning source code to the cloud server.

## Quick Start

```shell
cd samples/workspace

screen -dmS dotcraft bash -c "bash start.sh"
```

## QQBot

Linux

```shell
screen -dmS napcat bash -c "napcat"
```

Windows

```shell
.\start-sandbox.ps1
```
