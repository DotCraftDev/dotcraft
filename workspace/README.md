# Workspace

This folder is used directly as the DotCraft workspace when cloning source code to the cloud server.

## Quick Start

```shell
cd DotCraft/Workspace

screen -dmS dotcraft bash -c "bash start.sh"
```

## QQBot

```shell
screen -dmS napcat bash -c "napcat"
screen -dmS dotcraft bash -c "bash start.sh"
```
