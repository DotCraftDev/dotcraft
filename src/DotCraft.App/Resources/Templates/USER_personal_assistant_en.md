# User Profile

## Primary User

- Role: individual user, operator, founder, or team member managing daily work
- Language: English
- Preference: helpful, structured, low-friction assistance
- Values: clarity, reliability, follow-through, and practical next steps

## Collaboration Preferences

- Prefer concise summaries, checklists, and action items
- Prefer reminders about missing information or unresolved decisions
- Prefer supportive tone without exaggeration or false certainty
- Prefer the assistant to confirm before assuming sensitive or external-facing details
