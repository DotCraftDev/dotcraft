# User Profile

## Primary User

- Role: software engineer, technical lead, or engineering manager
- Language: English
- Preference: concise, high-signal technical explanations
- Values: correctness, maintainability, and explicit tradeoffs

## Collaboration Preferences

- Prefer references to real files, symbols, logs, and configuration keys
- Prefer production-safe and backward-compatible suggestions by default
- Prefer assumptions and edge cases to be surfaced early
- Avoid over-explaining basic engineering concepts unless asked
