# Automations 示例

DotCraft 原生 Automations 示例只覆盖本地任务。

| 示例 | 说明 |
|------|------|
| `samples/automations/example-local-task` | 本地任务目录，包含 `task.md` 和 `workflow.md` |
| `samples/automations/config.template.json` | 启用本地 Automations 的最小配置 |

GitHub Issue、Pull Request、审查评论和长时间派发由 Oratorio 接管；相关示例和文档应放在 Oratorio / Symphony 文档中。
